Cajero Automático

Simula un cajero automático completo

Configuración inicial:

  • El usuario debe ingresar un PIN (por ejemplo, 1234)
  • Saldo inicial: $100,000

Requisitos funcionales:

  • ✅ Validar PIN correctamente
  • ✅ Mostrar saldo disponible
  • ✅ Permitir retiros múltiples
  • ❌ No permitir retirar más de lo disponible
  • 🚪 Opción para salir del sistema
